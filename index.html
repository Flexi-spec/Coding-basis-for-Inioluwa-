HTML (HyperText Markup Language) is the backbone of a webpage. It tells the browser what content is on the page. It is NOT a programming language, just a markup language.

2. CSS ‚Äì The Style

CSS (Cascading Style Sheets) controls how HTML elements look. It‚Äôs like painting the walls, choosing furniture, and decorating the house. CSS makes your webpage visually appealing.

What CSS can do:

  • Change colors, fonts, and sizes
  • Adjust spacing, margins, and padding
  • Arrange layouts using Flexbox or Grid
  • Add animations and hover effects
  • Make pages responsive for mobile and desktop

Example CSS code:

h1 {
  color: purple;
  text-align: center;
  font-size: 36px;
}
p {
  color: #555;
  line-height: 1.5;
}

üí° Tip: You can write CSS inline, in a <style> tag, or in a separate .css file. The separation of style from structure keeps your code clean.

3. JavaScript ‚Äì The Interactivity

JavaScript (JS) makes webpages interactive. It responds to clicks, typing, moving the mouse, or other user actions.

Think of JS as electricity and smart systems in a house: it turns lights on, opens doors, and makes the environment react.

What JavaScript can do:

  • Show alerts and messages
  • Update content without refreshing the page
  • Validate forms and user input
  • Animate elements
  • Fetch data from servers (APIs)

Example JavaScript code:

function greet() {
  alert("Hello Inioluwa! Welcome to coding!");
}
greet();

üí° Tip: JS works with the DOM (Document Object Model), which represents your HTML elements as objects you can manipulate.
